# Milano Visual Browser

Prototipo interattivo mobile-first per esplorare monumenti di Milano con una logica da visual browser.

## Cosa include

- Mappa visuale di Milano con hotspot cliccabili
- Schede monumento con vista esterno / interno
- Transizioni, animazioni, scan AI e tour automatico
- Layer Web3 demo: wallet, passport, proof hash locale
- Deploy statico tramite GitHub Pages Actions

## Deploy

Il workflow `.github/workflows/pages.yml` pubblica il contenuto del repository su GitHub Pages.

URL atteso dopo l'attivazione Pages:

```txt
https://tronkss27.github.io/Test_milano_sito-/
```

Se il workflow non parte, apri GitHub → repository → Settings → Pages → Source → GitHub Actions.
