# Attivazione GitHub Pages

Il sito statico è già pronto nella root del repository.

Per pubblicarlo:

1. Apri Settings del repository.
2. Vai su Pages.
3. In Build and deployment scegli Source: Deploy from a branch.
4. Seleziona Branch: main.
5. Seleziona Folder: / root.
6. Salva.

URL atteso:

https://tronkss27.github.io/Test_milano_sito-/

Nota: il prototipo non richiede build, framework o server. GitHub Pages può servire direttamente index.html dalla branch main.
